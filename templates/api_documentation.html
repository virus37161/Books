Документация по использованию АПИ

Get запросы

http://127.0.0.1:8000/api/books/ - получение всех книг
Фильтрация http://127.0.0.1:8000/api/books/?id=1&category=1"
http://127.0.0.1:8000/api/authors/ - получение всех авторов
Фильтрация http://127.0.0.1:8000/api/authors/?id=&name=Джек Лондон
http://127.0.0.1:8000/api/category/ - получение списка категорий

Post запросы

Для Post запросов необходимо наличие токена, который можно получить по ссылке http://127.0.0.1:8000/api/token/

Создание книги:
JSON cхема:
data = {
"name": "Название книги",
"discription": "описание книги",
"category": [{"name_category": id_category}]
